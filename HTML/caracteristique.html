Qu'est-ce que DaVinci2020?

DaVinci2020 est un jeu de tire à la 3eme personne de type « battle royal ».
Le but de ce jeu est simple ; Être le dernier survivants des 100 joueurs
déployer sur l’une des 3 cartes intervertie toutes les semaines.
Venez a bout de vos ennemies à l’aide de diverses armes plus destructrices les unes que les autres.
Passer des soirées endiablées avec vos amis sur le mode multijoueur jouable jusqu’à 4 personnes.
Mais, ne vous en faite pas si vous voulez jouer tous seul, un mode de jeu solo est disponibles.

Vous voulez vous faire de nouveaux amis sur le jeu ?
C’est possible grâce a l’option « combler » disponible dans les modes ;
« Duo », « Trio » et dans le mode « section ».
daVinci est un jeu disponible sur toutes les plateformes ;
soit Xbox, Ps4, Pc, Téléphone et sur nintendo Switch .

De plus le jeu est disponible gratuitement
Des évènement a intervalles fréquents en liens avec nos autres jeux ou alors lié à des partenariats avec de grandes licences.
Le jeu propose aussi des tournois ouvert à tous pour les débutants comme pour les joueurs les plus aguerris.
A la clé de certains tournois une somme d’argents peut être versée au gagnant ou à son équipe.
